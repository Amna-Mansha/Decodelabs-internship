# AI Project 1 - Python Chatbot

## Description
This project is a simple chatbot developed using Python. The chatbot interacts with users by asking their name and allowing them to enter messages. It provides basic responses and continues the conversation until the user chooses to exit.

## Features
- User name input
- Basic conversation
- Continuous chat using a loop
- Exit option for ending the conversation

## Technologies Used
- Python

## How to Run
1. Install Python.
2. Open the project in VS Code.
3. Run the chatbot.py file.
4. Start chatting with the bot.
