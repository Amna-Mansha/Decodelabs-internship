# AI Recommendation Logic System

## Project Overview
This project is a simple AI Recommendation System that recommends courses based on user interests. The system uses similarity logic and pattern matching to suggest the most relevant items.

## Features
- Takes user interests as input
- Matches preferences with available items
- Calculates similarity score
- Displays recommendations

## Technologies Used
- Python
- Lists
- Dictionaries
- Sets

## How It Works
1. User enters interests.
2. System compares interests with item tags.
3. Match score is calculated.
4. Best matching items are recommended.

## AI Concept
Recommendation Systems using Similarity Logic and Pattern Matching.
